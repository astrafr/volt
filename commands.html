📜 Command List

Here’s what Volt can do! From fun little games to serious moderation tools, we’ve got you covered.

🎉 Fun Commands

  • ,compliment – Send someone a nice message
  • ,eightball – Ask the magic 8-ball a question
  • ,emoji – Get a random emoji or react to something
  • ,flip – Flip a coin
  • ,hacker – Fake “hacks” for laughs
  • ,joke – Get a random joke
  • ,mathmeme – Returns a funny math meme
  • ,rate – Rate anything from 1 to 10
  • ,rizz – Show off your rizz
  • ,roast – Roast someone (playfully!)
  • ,roll – Roll a dice
  • ,rps – Play rock-paper-scissors
  • ,saydumb – Make the bot say something silly
  • ,spamemoji – Spam emojis in a fun way
  • ,to_gif – Convert an uploaded image to a GIF

🛡️ Moderation Commands

  • ,ban – Ban a member from the server
  • ,clear – Clear a number of messages
  • ,kick – Kick a user
  • ,lock – Lock the current channel
  • ,mute – Mute a user
  • ,say – Let the bot say something
  • ,slowmode – Set slowmode on a channel
  • ,unban – Unban someone
  • ,unlock – Unlock a locked channel
  • ,unmute – Unmute a user

📦 Miscellaneous

  • ,help – Shows this help message

Pro tip: Use ,help <command> to get details about how to use each command!

You can also use ,help fun or ,help moderation for a category overview.
